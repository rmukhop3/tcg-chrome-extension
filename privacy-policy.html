Privacy Policy

Triangulator Chrome Extension

Overview

The Triangulator Chrome Extension ("the Extension") is developed by Arizona State University (ASU) to assist users in viewing course descriptions and finding equivalent ASU courses while browsing transfer guide websites. We are committed to protecting your privacy and being transparent about the data we collect and how we use it.

Information We Collect

The Extension collects and processes the following types of information:

Data You Provide

  • API Token: You provide a CreateAI API token to enable course lookup functionality. This token is stored locally in your browser.
  • Extension Settings: Your preferences (such as environment selection and display options) are stored locally in your browser.

Data Collected Automatically

  • Course Information: When you interact with course elements on supported websites, the Extension reads course identifiers (institution, subject, course number, and title) from the page to perform lookups.
Important: The Extension does NOT collect personal information such as your name, email address, browsing history outside of supported transfer guide websites, or any other personally identifiable information.

How We Use Information

The information collected is used solely for the following purposes:

  • To authenticate with the ASU CreateAI API service
  • To retrieve course descriptions from indexed academic catalogs
  • To find and display equivalent ASU courses
  • To remember your extension preferences

Data Storage

All user data (API token and settings) is stored locally in your browser using Chrome's chrome.storage.local API. This data:

  • Never leaves your device except when making authorized API requests
  • Is not synced across devices
  • Is automatically deleted when you uninstall the Extension
  • Can be cleared at any time through the Extension options page

Data Sharing

The Extension communicates with ASU's CreateAI API servers to retrieve course information. When making API requests, the following data is transmitted:

  • Your API authentication token
  • Course search queries (institution name, subject code, course number)

We do not sell, trade, or otherwise transfer your information to outside parties. Data transmitted to ASU servers is used only to fulfill course lookup requests.

Third-Party Services

The Extension interacts with the following ASU services:

  • CreateAI API (POC): api-main-poc.aiml.asu.edu
  • CreateAI API (Production): api-main.aiml.asu.edu

These services are operated by Arizona State University and are subject to ASU's institutional data protection policies.

Data Security

We implement appropriate security measures to protect your information:

  • All API communications use HTTPS encryption
  • API tokens are stored securely in browser local storage
  • No sensitive data is logged or stored on external servers

Your Rights

You have the right to:

  • Access: View your stored settings through the Extension options page
  • Modify: Change your API token and preferences at any time
  • Delete: Remove all stored data by uninstalling the Extension or clearing the data through browser settings
  • Opt-out: Disable or remove the Extension at any time

Permissions Explained

The Extension requires the following browser permissions:

  • activeTab: To read course information from the current page when you interact with course elements
  • storage: To save your API token and preferences locally
  • Host permissions: To communicate with ASU CreateAI API servers and supported transfer guide websites

Children's Privacy

The Extension is intended for use by college students and academic advisors. We do not knowingly collect information from children under 13 years of age.

Changes to This Policy

We may update this Privacy Policy from time to time. Any changes will be reflected in the "Last Updated" date below. We encourage you to review this policy periodically.
